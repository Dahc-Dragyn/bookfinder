# **App Name**: LibrarianAI

## Core Features:

- ISBN Book Data Lookup: Retrieve book data from Google Books and Open Library using ISBN, and display in a merged view.
- Hybrid Search: Allow users to search for books across multiple sources (Google Books, Open Library) and combine/deduplicate results.
- New Release Discovery: Provide a curated list of new book releases, aggregated from multiple sources.
- Author Details: Fetch and display detailed information about an author from Open Library based on the author's key.
- Book Recommendations: Suggest related books based on the current book or author the user is viewing using AI.
- Smart Summary: Generate concise summaries of book descriptions, extracting key points to provide a quick overview, using an AI tool

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke a sense of knowledge, mystery, and depth.
- Background color: Very light lavender (#F0F8FF, AliceBlue) as a calm, unobtrusive background.
- Accent color: Golden Yellow (#FFD700) to highlight important information and CTAs, adding a touch of sophistication and importance.
- Body and headline font: 'Literata', a serif font with a literary, vintage, slightly formal feel, is recommended.
- Use clear, minimalist icons sourced from a consistent set (e.g., Material Design Icons) for navigation and actions.
- Maintain a clean, card-based layout with consistent spacing and padding to present book information in an organized manner.
- Use subtle fade-in animations and transitions to enhance user experience without being distracting.
